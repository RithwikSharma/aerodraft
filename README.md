# AeroDraft

AI-assisted Type Investigation Report (TIR) drafting platform for MRO organisations.

Built for **SA Air Works, Gurgaon** — DGCA Part-145 approved.

**Live:** https://rithwiksharma.github.io/aerodraft/

## Stack
React + Vite · Vanilla CSS · OpenAI streaming API · localStorage persistence

## Features
- AI-generated TIRs in exact SA Air Works format
- Client registry with AD tracking
- Regulatory digest (DGCA · EASA · FAA)
- Report history with save/print
- Demo mode: GNC 255A avionics upgrade scenario
